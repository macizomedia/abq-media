# @abquanta/create-abq-module

Scaffold generator for new `@abquanta/*` modules using the shared template.

## Usage

From monorepo root:

```bash
node ./packages/create-abq-module/src/cli.js --name <module-name> --description "..."
```

Or helper script:

```bash
./scripts/create-module.sh <module-name> "description"
```
